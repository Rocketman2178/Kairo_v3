## Refinements Made

- **`languageService.ts` — `t()` function**: Changed `str.replace()` to `str.replaceAll()`. `String.prototype.replace` with a string pattern only replaces the first occurrence of a variable. If any translation string ever uses the same `{key}` twice, only the first would be substituted. `replaceAll` ensures all occurrences are replaced, future-proofing the function.
- **`PaymentFailedRecovery.tsx` — icon swap**: Replaced `PhoneCall` icon with `ArrowLeft` on the "Start a New Registration" button. `PhoneCall` implies calling someone; `ArrowLeft` correctly communicates navigation back to start.
