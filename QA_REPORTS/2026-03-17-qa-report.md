## Refinements Made
1. **`paymentPlans.ts` line 78 ‚Äî `buildDividedSchedule` first payment dueDate**: First installment incorrectly used `sessionStartDate.toISOString()` for its `dueDate` even though the label read "Due today". Fixed to `new Date().toISOString()`.
2. **`paymentPlans.ts` line 253 ‚Äî `formatPaymentOption` operator precedence**: `plan.savings ?? 0 / 100` was evaluated as `plan.savings ?? (0 / 100)` due to `??` having lower precedence than `/`. Savings amount was displayed in cents instead of dollars. Fixed to `(plan.savings ?? 0) / 100`. (Function is currently dead code ‚Äî not imported elsewhere ‚Äî but correction applied for correctness.)
3. **`useCartRecovery.ts` line 69 ‚Äî stale comment**: Comment said "matches DB expires_at" but the `abandoned_carts` table has no `expires_at` column. Removed the incorrect reference.
